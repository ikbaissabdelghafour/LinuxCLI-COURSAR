🔤 أوامر التعامل مع النصوص

Text Processing & Manipulation Commands

echo
طباعة
طباعة النص على الشاشة أو إلى ملف - Display text output
echo [OPTIONS] [STRING...]
الخيارات المتاحة:
-n
عدم إضافة سطر جديد في النهاية
-e
تفسير رموز الهروب مثل \n, \t
-E
عدم تفسير رموز الهروب (افتراضي)
رموز الهروب الشائعة:
\n سطر جديد
\t تبويب
\\ شرطة مائلة عكسية
\a جرس تنبيه
أمثلة عملية:
echo "Hello World"
echo -e "Line 1\nLine 2\tTabbed"
echo -n "No newline"
echo $USER $HOME
echo "Today is $(date)" > today.txt
grep
بحث
البحث عن النصوص في الملفات باستخدام التعبيرات النمطية - Search text patterns in files
grep [OPTIONS] PATTERN [FILE...]
الخيارات المتاحة:
-i
تجاهل حالة الأحرف (كبيرة/صغيرة)
-v
عكس البحث (إظهار الأسطر التي لا تطابق)
-n
عرض أرقام الأسطر
-c
عدد الأسطر المطابقة فقط
-l
أسماء الملفات التي تحتوي على النمط
-r
البحث في المجلدات بشكل تعاودي
-A NUM
عرض NUM أسطر بعد المطابقة
-B NUM
عرض NUM أسطر قبل المطابقة
-C NUM
عرض NUM أسطر قبل وبعد المطابقة
-E
استخدام التعبيرات النمطية الموسعة
-F
البحث عن نص ثابت (بدون regex)
--color
تلوين النتائج المطابقة
أنماط التعبيرات النمطية الشائعة:
^pattern النمط في بداية السطر
pattern$ النمط في نهاية السطر
. أي حرف واحد
* تكرار صفر أو أكثر
[abc] أي حرف من a أو b أو c
[0-9] أي رقم
أمثلة عملية:
grep -i "error" /var/log/syslog
grep -n "function" *.py
grep -r "TODO" /home/user/project/
grep -v "^#" config.conf
grep -E "^[0-9]+$" numbers.txt
sed
تعديل
محرر تيار للتعديل المتقدم للنصوص - Stream editor for text manipulation
sed [OPTIONS] 'SCRIPT' [FILE...]
الخيارات المتاحة:
-e
إضافة أوامر إلى النص البرمجي
-i
تعديل الملف في مكانه
-n
منع الطباعة التلقائية
-r
استخدام التعبيرات النمطية الموسعة
-f
قراءة النص البرمجي من ملف
الأوامر الأساسية في sed:
s/old/new/ استبدال النص
d حذف السطر
p طباعة السطر
i\text إدراج نص قبل السطر
a\text إضافة نص بعد السطر
أمثلة عملية:
sed 's/old/new/g' file.txt
sed -i 's/localhost/127.0.0.1/g' config.conf
sed '1,5d' file.txt
sed -n '10,20p' large_file.txt
sed '/pattern/d' input.txt
awk
معالجة
معالجة متقدمة للنصوص والأعمدة - Advanced text and column processing
awk [OPTIONS] 'PROGRAM' [FILE...]
الخيارات المتاحة:
-F
تحديد فاصل الحقول
-v
تعيين متغيرات
-f
قراءة البرنامج من ملف
-W
تفعيل التحذيرات
المتغيرات المدمجة في awk:
$0 السطر كاملاً
$1, $2, $3 الحقول الأول والثاني والثالث
NF عدد الحقول في السطر
NR رقم السطر الحالي
FS فاصل الحقول
أمثلة عملية:
awk '{print $1, $3}' data.txt
awk -F: '{print $1}' /etc/passwd
awk 'NR>1 {sum+=$2} END {print sum}' numbers.txt
awk '/pattern/ {print NR, $0}' file.txt
awk 'length($0) > 80' long_lines.txt
sort
ترتيب
ترتيب الأسطر أبجدياً أو رقمياً - Sort lines of text
sort [OPTIONS] [FILE...]
الخيارات المتاحة:
-r
ترتيب عكسي
-n
ترتيب رقمي
-k
ترتيب حسب عمود معين
-t
تحديد فاصل الحقول
-u
حذف الأسطر المتكررة
-f
تجاهل حالة الأحرف
-h
ترتيب الأحجام (1K, 2M, 3G)
-R
ترتيب عشوائي
أمثلة عملية:
sort names.txt
sort -n -r numbers.txt
sort -k2 -t: /etc/passwd
sort -u duplicate_lines.txt
ls -la | sort -k5 -h
uniq
تنظيف
إزالة أو عرض الأسطر المتكررة - Remove or report duplicate lines
uniq [OPTIONS] [INPUT [OUTPUT]]
الخيارات المتاحة:
-c
عرض عدد التكرارات
-d
عرض الأسطر المتكررة فقط
-u
عرض الأسطر الفريدة فقط
-i
تجاهل حالة الأحرف
-f
تجاهل عدد من الحقول الأولى
-s
تجاهل عدد من الأحرف الأولى
أمثلة عملية:
sort data.txt | uniq
sort names.txt | uniq -c
sort logs.txt | uniq -d
uniq -i mixed_case.txt
tr
تحويل
تحويل أو حذف الأحرف من الإدخال - Translate or delete characters
tr [OPTIONS] SET1 [SET2]
الخيارات المتاحة:
-d
حذف الأحرف الموجودة في SET1
-s
استبدال التكرارات المتتالية بحرف واحد
-c
عكس مجموعة الأحرف
أمثلة عملية:
echo "linux" | tr 'a-z' 'A-Z'
echo "122333" | tr -s '3'
echo "hello 123" | tr -d '0-9'
echo "abc" | tr -c 'a-c' '_'
wc
إحصاء
عد الكلمات أو الأسطر أو الأحرف في الملفات - Count lines, words, or bytes
wc [OPTIONS] [FILE...]
الخيارات المتاحة:
-l
عد الأسطر فقط
-w
عد الكلمات فقط
-c
عد البايتات فقط
-m
عد الأحرف فقط
أمثلة عملية:
wc file.txt
wc -l file.txt
wc -w *.txt
cat file.txt | wc -m
cut
تقطيع
استخراج الأعمدة أو الحقول من النصوص - Extract sections from lines of files
cut [OPTIONS] [FILE...]
الخيارات المتاحة:
-d
تحديد فاصل الحقول
-f
تحديد رقم الحقل أو الأعمدة
-c
تحديد نطاق الأحرف
--complement
عكس التحديد (إظهار ما لم يتم تحديده)
أمثلة عملية:
cut -d: -f1 /etc/passwd
cut -c1-5 file.txt
echo "a,b,c" | cut -d, -f2
cut --complement -c1-3 file.txt
