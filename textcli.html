🔤 أوامر معالجة النصوص

Text Processing Commands

echo
عرض
طباعة النص على الشاشة - Print text to screen
echo [OPTIONS] [STRING]
الخيارات المتاحة:
-n
عدم طباعة سطر جديد في النهاية
-e
تفعيل تفسير الرموز الخاصة مثل \n
أمثلة عملية:
echo "Hello World"
echo -e "Line 1\nLine 2"
grep
بحث
البحث عن نص في الملفات - Search text in files
grep [OPTIONS] PATTERN [FILE...]
الخيارات المتاحة:
-i
تجاهل حالة الأحرف
-r
بحث متكرر في المجلدات
-v
عرض الأسطر التي لا تطابق النمط
-n
عرض أرقام الأسطر
أمثلة عملية:
grep "error" /var/log/syslog
grep -r "TODO" .
sed
تعديل
محرر تيار للتعديل المتقدم للنصوص - Stream editor for text manipulation
sed [OPTIONS] 'SCRIPT' [FILE...]
الخيارات المتاحة:
-e
إضافة أوامر إلى النص البرمجي
-i
تعديل الملف في مكانه
-n
منع الطباعة التلقائية
-r
استخدام التعبيرات النمطية الموسعة
-f
قراءة النص البرمجي من ملف
الأوامر الأساسية في sed:
s/old/new/ استبدال النص
d حذف السطر
p طباعة السطر
i\text إدراج نص قبل السطر
a\text إضافة نص بعد السطر
أمثلة عملية:
sed 's/old/new/g' file.txt
sed -i 's/localhost/127.0.0.1/g' config.conf
sed '1,5d' file.txt
sed -n '10,20p' large_file.txt
sed '/pattern/d' input.txt
awk
معالجة
معالجة متقدمة للنصوص والأعمدة - Advanced text and column processing
awk [OPTIONS] 'PROGRAM' [FILE...]
الخيارات المتاحة:
-F
تحديد فاصل الحقول
-v
تعيين متغيرات
-f
قراءة البرنامج من ملف
-W
تفعيل التحذيرات
المتغيرات المدمجة في awk:
$0 السطر كاملاً
$1, $2, $3 الحقول الأول والثاني والثالث
NF عدد الحقول في السطر
NR رقم السطر الحالي
FS فاصل الحقول
أمثلة عملية:
awk '{print $1, $3}' data.txt
awk -F: '{print $1}' /etc/passwd
awk 'NR>1 {sum+=$2} END {print sum}' numbers.txt
awk '/pattern/ {print NR, $0}' file.txt
awk 'length($0) > 80' long_lines.txt
sort
ترتيب
ترتيب الأسطر أبجدياً أو رقمياً - Sort lines of text
sort [OPTIONS] [FILE...]
الخيارات المتاحة:
-r
ترتيب عكسي
-n
ترتيب رقمي
-k
ترتيب حسب عمود معين
-t
تحديد فاصل الحقول
-u
حذف الأسطر المتكررة
-f
تجاهل حالة الأحرف
-h
ترتيب الأحجام (1K, 2M, 3G)
-R
ترتيب عشوائي
أمثلة عملية:
sort names.txt
sort -n -r numbers.txt
sort -k2 -t: /etc/passwd
sort -u duplicate_lines.txt
ls -la | sort -k5 -h
uniq
تنظيف
إزالة أو عرض الأسطر المتكررة - Remove or report duplicate lines
uniq [OPTIONS] [INPUT [OUTPUT]]
الخيارات المتاحة:
-c
عرض عدد التكرارات
-d
عرض الأسطر المتكررة فقط
-u
عرض الأسطر الفريدة فقط
-i
تجاهل حالة الأحرف
-f
تجاهل عدد من الحقول الأولى
-s
تجاهل عدد من الأحرف الأولى
أمثلة عملية:
sort data.txt | uniq
sort names.txt | uniq -c
sort logs.txt | uniq -d
uniq -i mixed_case.txt
tr
تحويل
تحويل أو حذف الأحرف من الإدخال - Translate or delete characters
tr [OPTIONS] SET1 [SET2]
الخيارات المتاحة:
-d
حذف الأحرف الموجودة في SET1
-s
استبدال التكرارات المتتالية بحرف واحد
-c
عكس مجموعة الأحرف
أمثلة عملية:
echo "linux" | tr 'a-z' 'A-Z'
echo "122333" | tr -s '3'
echo "hello 123" | tr -d '0-9'
echo "abc" | tr -c 'a-c' '_'
wc
إحصاء
عد الكلمات أو الأسطر أو الأحرف في الملفات - Count lines, words, or bytes
wc [OPTIONS] [FILE...]
الخيارات المتاحة:
-l
عد الأسطر فقط
-w
عد الكلمات فقط
-c
عد البايتات فقط
-m
عد الأحرف فقط
أمثلة عملية:
wc file.txt
wc -l file.txt
wc -w *.txt
cat file.txt | wc -m
cut
تقطيع
استخراج الأعمدة أو الحقول من النصوص - Extract sections from lines of files
cut [OPTIONS] [FILE...]
الخيارات المتاحة:
-d
تحديد فاصل الحقول
-f
تحديد رقم الحقل أو الأعمدة
-c
تحديد نطاق الأحرف
--complement
عكس التحديد (إظهار ما لم يتم تحديده)
أمثلة عملية:
cut -d: -f1 /etc/passwd
cut -c1-5 file.txt
echo "a,b,c" | cut -d, -f2
cut --complement -c1-3 file.txt
paste
دمج
دمج الأسطر من ملفات متعددة جنباً إلى جنب - Merge lines of files horizontally
paste [OPTIONS] [FILE...]
الخيارات المتاحة:
-d
تحديد فاصل الأعمدة (افتراضي: Tab)
-s
دمج جميع الأسطر من كل ملف في سطر واحد
أمثلة عملية:
paste file1.txt file2.txt
paste -d, col1.txt col2.txt
paste -s file.txt
join
ربط
ربط الأسطر من ملفين بناءً على حقل مشترك - Join lines of two files on a common field
join [OPTIONS] FILE1 FILE2
الخيارات المتاحة:
-1 FIELD
تحديد رقم الحقل من الملف الأول للربط
-2 FIELD
تحديد رقم الحقل من الملف الثاني للربط
-t CHAR
تحديد فاصل الحقول (افتراضي: مسافة)
-o FORMAT
تحديد تنسيق الإخراج
-a FILENUM
عرض الأسطر غير المطابقة من ملف معين
-e STRING
استبدال الحقول الفارغة بسلسلة نصية
أمثلة عملية:
join file1.txt file2.txt
join -1 2 -2 1 a.txt b.txt
join -t, data1.csv data2.csv
join -a 1 file1.txt file2.txt
join -o 1.1,2.2 file1.txt file2.txt
