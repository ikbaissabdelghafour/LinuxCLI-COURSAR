🔐 أوامر الصلاحيات و المِلكية

Permissions & Ownership Commands

chmod
تغيير الصلاحيات
تغيير صلاحيات الملفات أو المجلدات - Change file/directory permissions
chmod [OPTIONS] MODE FILE
الخيارات المتاحة:
-R
تغيير الصلاحيات بشكل递归 للمجلدات الفرعية
--reference=FILE
استخدام صلاحيات ملف آخر كمرجع
أمثلة عملية:
chmod 755 script.sh
chmod -R 644 myfolder
chmod u+x file.sh
chown
تغيير المالك
تغيير مالك الملف أو المجلد - Change file/directory owner
chown [OPTIONS] OWNER[:GROUP] FILE
الخيارات المتاحة:
-R
تغيير المالك بشكل递归 للمجلدات الفرعية
--reference=FILE
استخدام مالك ملف آخر كمرجع
أمثلة عملية:
chown user file.txt
chown user:group file.txt
chown -R user myfolder
chgrp
تغيير المجموعة
تغيير مجموعة الملف أو المجلد - Change file/directory group
chgrp [OPTIONS] GROUP FILE
الخيارات المتاحة:
-R
تغيير المجموعة بشكل递归 للمجلدات الفرعية
--reference=FILE
استخدام مجموعة ملف آخر كمرجع
أمثلة عملية:
chgrp staff file.txt
chgrp -R users myfolder
umask
إعداد الصلاحيات الافتراضية
تعيين الصلاحيات الافتراضية للملفات الجديدة - Set default permissions for new files
umask [VALUE]
أمثلة عملية:
umask
umask 022
