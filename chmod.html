صلاحيات الملفات (File Permissions)
فـ لينكس الصلاحيات عندها رموز وحساب بالأرقام:
الرمز المعنى القيمة الرقمية مثال
r Read (قراءة) 4 قراءة الملف
w Write (كتابة) 2 تغيير/تعديل الملف
x Execute (تنفيذ) 1 تشغيل الملف (سكربت/برنامج)

طريقة الحساب:
كل صلاحية عندها رقم، وكنجمعو الأرقام باش نحددو الصلاحيات.
  • 7 = 4+2+1 = rwx (قراءة + كتابة + تنفيذ)
  • 6 = 4+2 = rw- (قراءة + كتابة)
  • 5 = 4+1 = r-x (قراءة + تنفيذ)
  • 4 = r-- (قراءة فقط)
  • 0 = --- (لا صلاحيات)

أمثلة عملية:
  • chmod 777 file.txt → كلشي عندو rwx (full access)
  • chmod 644 file.txt → المالك rw-، الباقي r--
  • chmod 755 script.sh → المالك rwx، الباقي r-x
