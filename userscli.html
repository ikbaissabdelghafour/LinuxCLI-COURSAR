👤 أوامر إدارة المستخدمين

User Management Commands

whoami
معلومات
عرض اسم المستخدم الحالي - Show current username
whoami
أمثلة عملية:
whoami
who
معلومات
عرض المستخدمين المتصلين حالياً - Show logged in users
who
أمثلة عملية:
who
id
معلومات
عرض UID, GID والمجموعات - Show UID, GID, groups
id [USER]
أمثلة عملية:
id
id username
adduser
إدارة
إضافة مستخدم جديد للنظام - Add a new user
adduser [OPTIONS] USERNAME
أمثلة عملية:
sudo adduser newuser
passwd
إدارة
تغيير كلمة مرور المستخدم - Change user password
passwd [USER]
أمثلة عملية:
passwd
sudo passwd username
su
إدارة
تبديل المستخدم الحالي - Switch user
su [USER]
أمثلة عملية:
su
su username
sudo
إدارة
تنفيذ أمر بصلاحيات root - Run command as root
sudo [COMMAND]
أمثلة عملية:
sudo apt update
sudo reboot
