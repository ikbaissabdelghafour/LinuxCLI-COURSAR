System & Process Management Commands

ps
عمليات
عرض العمليات الجارية - Show running processes
ps [OPTIONS]
أمثلة عملية:
ps aux
ps -ef
top
مراقبة
مراقبة الأداء (CPU, RAM) - Monitor system performance
top
أمثلة عملية:
top
htop
مراقبة
نسخة متطورة من top - Advanced interactive process viewer
htop
أمثلة عملية:
htop
kill
عمليات
إنهاء عملية باستخدام PID - Kill process by PID
kill [OPTIONS] PID
أمثلة عملية:
kill 1234
kill -9 1234
killall
عمليات
إنهاء جميع العمليات بنفس الاسم - Kill all processes by name
killall [OPTIONS] NAME
أمثلة عملية:
killall firefox
uptime
معلومات
مدة تشغيل النظام - Show system uptime
uptime
أمثلة عملية:
uptime
uname -a
معلومات
معلومات النظام - Show system information
uname -a
أمثلة عملية:
uname -a
df
مساحة
عرض المساحة المستعملة والفارغة - Show disk usage
df [OPTIONS]
أمثلة عملية:
df -h
du
مساحة
عرض حجم الملفات/المجلدات - Show file/folder size
du [OPTIONS] [PATH]
أمثلة عملية:
du -sh *
du -h /home/user
free
ذاكرة
عرض الرام المستعملة - Show RAM usage
free [OPTIONS]
أمثلة عملية:
free -h
