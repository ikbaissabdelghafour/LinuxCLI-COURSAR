📁 أوامر إدارة الملفات والمجلدات

File & Directory Management Commands

ls
عرض
عرض محتويات المجلد - List directory contents
ls [OPTIONS] [DIRECTORY]
الخيارات المتاحة:
-l
عرض مفصل مع الصلاحيات والحجم والتاريخ
-a
عرض جميع الملفات بما في ذلك المخفية
-h
أحجام قابلة للقراءة (KB, MB, GB)
-R
عرض المجلدات الفرعية بشكل تعاودي
-t
ترتيب حسب وقت التعديل (الأحدث أولاً)
-S
ترتيب حسب الحجم (الأكبر أولاً)
-r
عكس ترتيب العرض
-1
عرض ملف واحد في كل سطر
أمثلة عملية:
ls -la /home/user
ls -lh *.txt
ls -lt | head -10
ls -laS /var/log
cd
تنقل
التنقل بين المجلدات - Change directory
cd [DIRECTORY]
الاستخدامات الشائعة:
cd ~
العودة للمجلد الرئيسي للمستخدم
cd ..
الانتقال للمجلد الأب (المجلد الأعلى)
cd -
العودة للمجلد السابق
cd /
الانتقال لجذر النظام
cd
بدون معاملات - العودة للمجلد الرئيسي
أمثلة عملية:
cd /var/log
cd ~/Documents
cd ../../../
cd /etc/apache2/sites-available
mkdir
إنشاء
إنشاء مجلد جديد - Create directories
mkdir [OPTIONS] DIRECTORY...
الخيارات المتاحة:
-p
إنشاء المجلدات الفرعية إذا لم تكن موجودة
-m
تحديد الصلاحيات أثناء الإنشاء (مثل: 755)
-v
عرض رسالة تأكيد الإنشاء
أمثلة عملية:
mkdir -p /path/to/deep/directory
mkdir -m 755 new_folder
mkdir folder1 folder2 folder3
mkdir -pv ~/projects/{frontend,backend,database}
cp
نسخ
نسخ الملفات والمجلدات - Copy files and directories
cp [OPTIONS] SOURCE... DESTINATION
الخيارات المتاحة:
-r
نسخ المجلدات مع جميع محتوياتها
-i
السؤال قبل الكتابة فوق ملف موجود
-v
عرض تفاصيل عملية النسخ
-p
الحفاظ على الصلاحيات والتوقيت والمالك
-u
نسخ فقط إذا كان المصدر أحدث من الهدف
-a
نسخ أرشيفية (يحافظ على كل شيء)
-f
فرض النسخ بدون تأكيد
-L
تتبع الروابط الرمزية
أمثلة عملية:
cp -r /source/folder /destination/
cp -iv file1.txt backup/
cp *.jpg /images/
cp -a /home/user /backup/
mv
نقل
نقل أو إعادة تسمية الملفات والمجلدات - Move/rename files
mv [OPTIONS] SOURCE... DESTINATION
الخيارات المتاحة:
-i
السؤال قبل الكتابة فوق ملف موجود
-v
عرض تفاصيل عملية النقل
-n
عدم الكتابة فوق ملف موجود
-f
فرض النقل بدون تأكيد
-u
نقل فقط إذا كان المصدر أحدث
أمثلة عملية:
mv old_name.txt new_name.txt
mv *.pdf /documents/
mv -iv file.txt /backup/
mv /tmp/file /home/user/important/
rm
حذف
حذف الملفات والمجلدات - Remove files and directories
rm [OPTIONS] FILE...
الخيارات المتاحة:
-r
حذف المجلدات مع جميع محتوياتها
-f
حذف إجباري بدون تأكيد أو أخطاء
-i
السؤال قبل كل حذف
-v
عرض تفاصيل عملية الحذف
-I
التأكيد مرة واحدة عند حذف أكثر من 3 ملفات
--preserve-root
حماية جذر النظام من الحذف
أمثلة عملية:
rm -rf /tmp/old_folder
rm -i *.log
rm -v unwanted_file.txt
rm -I /path/to/many/files/*
find
بحث
البحث عن الملفات والمجلدات بمعايير متقدمة - Advanced file search
find [PATH] [OPTIONS] [EXPRESSION]
الخيارات والمعايير:
-name
البحث بالاسم (حساس للأحرف)
-iname
البحث بالاسم (غير حساس للأحرف)
-type f
البحث عن الملفات فقط
-type d
البحث عن المجلدات فقط
-size
البحث بالحجم (+10M, -1G, 50k)
-mtime
البحث بتاريخ التعديل (بالأيام)
-exec
تنفيذ أمر على النتائج
-delete
حذف النتائج المطابقة
أمثلة عملية:
find /home -name "*.txt"
find . -type f -size +10M
find /var/log -name "*.log" -mtime +7 -delete
find /etc -iname "*conf*" -type f
find . -name "*.tmp" -exec rm {} \;
locate
بحث سريع
البحث السريع باستعمال قاعدة البيانات المحدثة
locate [OPTIONS] PATTERN
الخيارات المتاحة:
-i
تجاهل حالة الأحرف أثناء البحث
-c
عرض عدد النتائج فقط
-r
استخدام تعبير نمطي (regular expression) للبحث
-e
عرض الملفات الموجودة فقط
أمثلة عملية:
locate passwd
locate -i readme
locate -c .conf
locate -r '\.log$'
locate -e /etc/hosts
