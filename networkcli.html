🌐 أوامر الشبكة

Network Commands

ping
اختبار
اختبار الاتصال بمضيف - Test connectivity to a host
ping [OPTIONS] HOST
أمثلة عملية:
ping google.com
ping -c 4 8.8.8.8
curl
جلب بيانات
جلب بيانات من عنوان URL - Fetch data from URL
curl [OPTIONS] URL
أمثلة عملية:
curl https://example.com
curl -I https://example.com
wget
تحميل
تحميل ملفات من الإنترنت - Download files from the internet
wget [OPTIONS] URL
أمثلة عملية:
wget https://example.com/file.zip
ifconfig
معلومات
عرض إعدادات الشبكة (قديم) - Show network configuration (legacy)
ifconfig
أمثلة عملية:
ifconfig
ip addr
معلومات
عرض إعدادات الشبكة (حديث) - Show network configuration (modern)
ip addr
أمثلة عملية:
ip addr
netstat
مراقبة
مراقبة المنافذ والاتصالات - Monitor ports and connections
netstat [OPTIONS]
أمثلة عملية:
netstat -tuln
netstat -anp
ssh
اتصال
الاتصال بسيرفر آخر عبر SSH - Connect to another server via SSH
ssh [USER@]HOST
أمثلة عملية:
ssh user@192.168.1.10
scp
نقل ملفات
نقل الملفات عبر SSH - Copy files over SSH
scp [OPTIONS] SRC DEST
أمثلة عملية:
scp file.txt user@host:/path/
scp -r folder user@host:/path/
