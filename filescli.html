📄 أوامر عرض وتعديل الملفات

File Viewing & Editing Commands

cat
عرض
عرض محتوى الملفات بالكامل - Display complete file content
cat [OPTIONS] [FILE...]
الخيارات المتاحة:
-n
ترقيم جميع الأسطر
-b
ترقيم الأسطر غير الفارغة فقط
-A
عرض جميع الرموز الخاصة والمخفية
-s
ضغط الأسطر الفارغة المتعددة لسطر واحد
-E
عرض رمز $ في نهاية كل سطر
-T
عرض رموز التبويب كـ ^I
أمثلة عملية:
cat -n /etc/passwd
cat file1.txt file2.txt > combined.txt
cat -A special_chars.txt
cat -s long_file.txt | less
less
عرض
عرض الملفات مع إمكانية التنقل والبحث - Interactive file viewer
less [OPTIONS] [FILE]
الخيارات المتاحة:
-N
عرض أرقام الأسطر
-S
قطع الأسطر الطويلة (عدم التفاف)
-i
تجاهل حالة الأحرف في البحث
-I
تجاهل حالة الأحرف نهائياً
+G
البدء من نهاية الملف
-F
الخروج إذا كان الملف يتسع في شاشة واحدة
اختصارات لوحة المفاتيح داخل less:
Space أو f الصفحة التالية
b الصفحة السابقة
/text البحث عن نص
n النتيجة التالية
N النتيجة السابقة
q الخروج
أمثلة عملية:
less -N /var/log/syslog
less +G large_file.txt
ps aux | less
less -S wide_data.csv
more
عرض
عرض المحتوى تدريجياً صفحة بصفحة - Simple page viewer
more [OPTIONS] [FILE]
الخيارات المتاحة:
-d
عرض رسائل مساعدة للمستخدم
-f
عدد الأسطر بدلاً من الحجم الفعلي
-p
عدم التمرير، مسح الشاشة بدلاً من ذلك
-s
ضغط الأسطر الفارغة المتعددة
-u
منع التسطير
+num
البدء من السطر رقم num
أمثلة عملية:
more /etc/services
more +50 large_file.txt
ls -la | more
more -s text_with_blanks.txt
head
عرض
عرض الأسطر الأولى من الملف - Display first lines of file
head [OPTIONS] [FILE]
الخيارات المتاحة:
-n NUM
عدد الأسطر المراد عرضها (افتراضي: 10)
-c NUM
عدد البايتات المراد عرضها
-v
عرض اسم الملف دائماً
-q
عدم عرض اسم الملف أبداً
-f
متابعة نمو الملف (مثل tail -f)
أمثلة عملية:
head -n 5 /etc/passwd
head -c 100 binary_file
head -v *.txt
head -20 /var/log/messages
tail
عرض
عرض الأسطر الأخيرة من الملف - Display last lines of file
tail [OPTIONS] [FILE]
الخيارات المتاحة:
-n NUM
عدد الأسطر المراد عرضها (افتراضي: 10)
-f
متابعة نمو الملف في الوقت الفعلي
-F
متابعة حتى لو تم إعادة إنشاء الملف
-c NUM
عدد البايتات المراد عرضها
-v
عرض اسم الملف دائماً
-q
عدم عرض اسم الملف أبداً
--retry
إعادة المحاولة إذا لم يكن الملف متاحاً
-s NUM
فترة التحديث بالثواني (مع -f)
أمثلة عملية:
tail -f /var/log/syslog
tail -n 50 /var/log/apache2/access.log
tail -c 200 data_file.bin
tail -F /var/log/application.log
nano
تحرير
محرر نصوص بسيط وسهل الاستخدام - Simple text editor
nano [OPTIONS] [FILE]
الخيارات المتاحة:
-l
عرض أرقام الأسطر
-m
تفعيل دعم الماوس
-i
التبويب التلقائي للأسطر الجديدة
-w
عدم التفاف الأسطر الطويلة
-x
عدم عرض شريط المساعدة
-E
تحويل التبويبات إلى مسافات
-S
تفعيل التمرير السلس
+NUM
البدء من السطر رقم NUM
اختصارات nano الأساسية:
Ctrl+O حفظ الملف
Ctrl+X الخروج
Ctrl+W البحث
Ctrl+K قص السطر
Ctrl+U لصق
Ctrl+G المساعدة
أمثلة عملية:
nano -l script.py
nano -w /etc/hosts
nano +25 large_file.txt
nano -m -i config.conf
vim
تحرير
محرر نصوص قوي ومتقدم - Advanced text editor
vim [OPTIONS] [FILE]
الخيارات المتاحة:
-r
استعادة ملف من نسخة احتياطية
-R
فتح في وضع القراءة فقط
-n
عدم استخدام ملف swap
-o
فتح الملفات في نوافذ أفقية
-O
فتح الملفات في نوافذ عمودية
-u FILE
استخدام ملف إعدادات مخصص
أوامر vim الأساسية:
i دخول وضع الإدخال
:w حفظ الملف
:q الخروج
:wq حفظ والخروج
dd حذف السطر الحالي
u تراجع
/text بحث عن نص
:help المساعدة
أمثلة عملية:
vim file.txt
vim -R readonly.txt
vim -o file1.txt file2.txt
vim -u custom_vimrc file.txt
