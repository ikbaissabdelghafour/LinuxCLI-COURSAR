📦 أوامر الحزم

Package Management Commands

apt-get / apt
Debian/Ubuntu
إدارة الحزم في توزيعات ديبيان وأوبونتو - Manage packages (Debian/Ubuntu)
apt [OPTIONS] COMMAND
أمثلة عملية:
sudo apt update
sudo apt install package
sudo apt remove package
yum / dnf
Fedora/RedHat
إدارة الحزم في فيدورا وريد هات - Manage packages (Fedora/RedHat)
yum [OPTIONS] COMMAND
أمثلة عملية:
sudo dnf update
sudo yum install package
sudo yum remove package
pacman
Arch Linux
إدارة الحزم في آرتش لينكس - Manage packages (Arch Linux)
pacman [OPTIONS] COMMAND
أمثلة عملية:
sudo pacman -Syu
sudo pacman -S package
sudo pacman -R package
