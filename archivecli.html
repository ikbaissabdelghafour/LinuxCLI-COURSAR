🗜️ أوامر الأرشفة و الضغط

Archiving & Compression Commands

tar
أرشفة
أرشفة الملفات والمجلدات - Archive files and folders
tar [OPTIONS] ARCHIVE FILES
أمثلة عملية:
tar -cvf archive.tar folder/
tar -xvf archive.tar
tar -czvf archive.tar.gz folder/
gzip
ضغط
ضغط الملفات باستخدام gzip - Compress files with gzip
gzip [OPTIONS] FILE
أمثلة عملية:
gzip file.txt
gzip -d file.txt.gz
gunzip
فك ضغط
فك ضغط ملفات gzip - Decompress gzip files
gunzip [OPTIONS] FILE.gz
أمثلة عملية:
gunzip file.txt.gz
zip
ضغط
ضغط الملفات إلى صيغة zip - Compress files to zip format
zip [OPTIONS] ARCHIVE FILES
أمثلة عملية:
zip archive.zip file1 file2
zip -r archive.zip folder/
unzip
فك ضغط
فك ضغط ملفات zip - Extract zip files
unzip [OPTIONS] ARCHIVE.zip
أمثلة عملية:
unzip archive.zip
unzip -l archive.zip
